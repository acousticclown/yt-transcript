# v1.2.0 - AI-Powered Notes Application Plan

## Vision

Transform YT-Transcript into a **personalized AI-powered notes application** with YouTube transcript integration. A modern, clean design inspired by Notion/Linear aesthetics.

---

## Design System (from screenshots)

### Color Palette

- **Background:** Warm cream/beige (`#F5F3EF`, `#FAF9F7`)
- **Surface:** Pure white with subtle shadows
- **Primary Accent:** Golden yellow/amber (`#F5A623`, `#FFBA08`)
- **Secondary:** Soft green (`#4A7C59`, `#6B9080`)
- **Text Primary:** Dark charcoal (`#1A1A1A`)
- **Text Secondary:** Warm gray (`#6B6B6B`)
- **Borders:** Very subtle (`#E8E6E1`)

### Typography

- **Font:** Clean sans-serif (SF Pro, Inter, or similar)
- **Headings:** Bold, large (24-32px)
- **Body:** Regular, readable (14-16px)

### Design Principles

- Warm, inviting color palette (not cold blue/gray)
- Generous whitespace
- Soft shadows, no harsh borders
- Card-based layout with rounded corners (12-16px)
- Sidebar navigation pattern
- Clean iconography

## Tech Decisions

- **Routing:** Next.js App Router with route groups
- **State:** React Context for user/notes (no external state lib yet)
- **Editor:** Start simple (textarea), upgrade to Tiptap/ProseMirror later
- **Storage:** Local JSON files initially (existing pattern)
- **Auth:** Mock context, real auth in v1.3.0

---

## Success Criteria

1. Clean, warm, modern UI matching design references
2. Smooth navigation between landing → auth → dashboard
3. Notes can be created, edited, organized
4. YT transcript can be imported as notes
5. Mobile-first, works great on all devices
6. AI features enhance note-taking experience
